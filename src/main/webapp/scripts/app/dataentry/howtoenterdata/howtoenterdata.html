How to enter data ?



Data must be previously compiled in a dedicated Excel file (one article/Excel file).

Important

  • When opening the Excel file, answer YES to the start-up question 'enable the macros'.If not, you will not be able to enter data.
  • Do not change the format of the cells (only the contents of the cells can be modified).
  • Strictly respect the instructions from the yellow pop-up.
  • If a cell is orange, it must be filled.
  • Do not 'jump over' lines (no empty line between 2 lines).
  • Columns with the suffix * are mandatory.
1 - PUBLI thumb

This is the starting point of the database, interconnected with other thumbs.
It is very important to respect the popup instructions.
To make the compilation easier it is recommended to export the bibliography from MpDBc citation style for Zotero. Download the MpDBc citation style and add it in your Zotero preferences.

Entry type: with the scroll down button, choose a document type.
Author: copy paste the list of authors from the Zotero MPDBC edited bibliography.
Title: copy-paste from the Zotero MPDBC edited bibliography.
journal: use entire name of the journal, example 'Journal of Ethnopharmacology' no abbreviation as 'J Ethnopharmacol'

The other columns are filled if data are available.

Compiler name: full name of compiler (the person who has entered all data) must be noted. Last name first (family name), first name then (example: Fonda, Jane).
Note from compiler: after entering data, the compiler should make a brief appreciative comment on the scientific quality of the publication (this is optional).

2 - SPECIES sheet

Publication title: use the scroll down list or copy-paste the publication title.
Species name in publication: write (copy-paste) the Latin binomial of species exactly as indicated in the publication.
Species: check the Latin binomial as written in publication on the Internet website The Plant List. Check if it is correctly spelled and valid. If not, enter correct actualized full Latin binomial (with authors name). If the Latin binomial is correct in the publication, repeat it in this column. If the Latin binomial as spelled in the publication does not exist in the PLANT LIST, DO NOT register this species.
Herbarium: if a herbarium sample has been collected and deposited in a herbarium, mark 'yes' and note the herbarium name if indicated.
The other columns are filled if data are available.

3 - PLANT INGREDIENTS sheet

Publication title: use the dropdown list or copy-paste the publication title.
Plants ingredient: write down species name (accepted full Latin binomial) and plant part(s) used in remedy (if reported in 4 - ETHNO sheet) or in evaluated extracts (if reported in 5 - IN VIVO PHARMACO and 6 - IN VITRO PHARMACO sheet).

4 - ETHNO sheet

Publication title: use the dropdown list or copy-paste the publication title.
Plant ingredient(s) used: fill according to previously defined species + plants parts (cf. 3 - Plant ingredients sheet).
Ethnopharmacological relevancy: note ethnobotanical use of species or remedy as quoted in the publication (do not create new lines if many uses are reported, just put them all in one cell). If ethnobotanical use originates from another publication (cited in the compiled publication), note the complete publication reference (copy-paste from the bibliography of the publication) in the same cell.

5 - IN VIVO PHARMACO sheet

Publication title: use the dropdown list or copy-paste the publication title.
Plant ingredient(s) used: fill up according previously defined species + plants parts (cf. 3 - Plant ingredients sheet).
Tested entity: use the dropdown list and select extract, fraction, or pure compound. If pure compound, write the name (Compound name column).
Extract solvent: in the case of an extract, use the dropdown list to select the solvent used for the extraction (if reported).
Additive product: note the name of the product possibly added to entity tested.
Screening test: use the dropdown list (see details below for the choice of the antimalarial test).
Note on the choice Toxicity in the dropdown list Screening test column:

  • If an in vivo toxicity test has been performed, tick the Toxicity choice in the dropdown menu and fill the LD50 column (if reported).
  • If an in vivo toxicity test has been performed but different from a LD50, tick the Toxicity choice in the in the dropdown menu. Do not fill another column.
Note that the selected result is the best result obtained for a given Screening test and for a selection of Plant(s) ingredient(s) + Tested entity (extract/fraction/compound) + a given Treatment route.

According to the best result, fill the next columns if available: Dose for selected result, Inhibition for selected result, Survival percent for selected result, Survival time for selected result and ED50.
Experimental observation from compiler(s): free column, to be filled if necessary by compiler with any relevant information about experimental procedure, analysis of results, etc.

6 - IN VITRO PHARMACO sheet

Publication title: use the dropdown list or copy-paste the publication title.
Plant ingredient(s) used column: fill according to previously defined species + plants parts (cf. 3 - Plant ingredients sheet).
Tested entity: note if it's an extract, a fraction, or a pure compound. If pure compound, write the name ('Compound name' column).
Extract solvent: in the case of an extract, use the dropdown list to select the solvent used for the extraction (if reported).
Additive product: note the name of the product possibly added to entity tested.
Screening test: see details below for the choice of the antimalarial test.
Note on the choice Toxicity in Screening test column:

  • If an in vitro toxicity test has been performed and a selectivity index has been calculated, tick the Toxicity choice in the dropdown menu and fill the column Selectivity Index for selected result.
  • If an in vitro toxicity test has been performed and a selectivity index has not been calculated, tick the Toxicity choice in the dropdown menu. Do not fill another column.
Note that selected result is the best result obtained in a given Screening test for a selection of Plant(s) ingredient(s) + Tested entity (extract/fraction/compound).

According to the result selected, fill the next columns if available: Concentration for selected result, Inhibition for selected result, and IC50 for selected result. The Measure method column can also be filled.
Note on the measure unit for in vitro results:

  • In case of an extract or fraction, results are given in µg/ml.
  • In case of a pure compound, results are given in µmole/ml.
Experimental observation from compiler(s): free column, to be filled if necessary by compiler with any relevant information about experimental procedure, analysis of results, etc.


Notes on screening tests

Protocols detailed below are standard references for specific screening tests and correspond to the dropdown menu of the Screening test column. The compiler must be aware that the protocol used in the paper must match with the protocols described below. If not, tick the 'Other' choice in the dropdown menu.

A. IN VIVO
1. Schizontocidal activity (activity on erythrocytic stages)

Peter’s test, suppressive test (Evaluation of schizontocidal activity on early infection) : Groups of mice are infected at D0, and then they are treated (IP, Oral, SC) from day 0 to day 3. The suppressive effects is estimated on day 4, examining Giemsa-stained thin blood smears made from the tail of the treated mice and compared with a control group of mice treated with the solvent.

Rane test, curative test (schizontocidal activity in established infection) : The 4-day test is modified so that the first treatment did not take place until 72 hours after the mice had been infected. Then mice are treated daily for x days and parasitaemias evaluated on each day of treatment. In addition, mortality is followed-up.

Repository activity/prophylactic activity, preventive activity : Mice are treated (with the drugs) pre-infection, for X consecutive days (days 0–X). On day X+1, the mice are inoculated with Plasmodium (as in the 4-day test, Peter’s test) and on day 7 (72 hours post-infection) their parasitaemias are assessed.

2. Transmission inhibition test

Gametocytes: The effect on the formation of gametocytes (sexual forms, found in the host blood, who reproduce in the mosquito gut) is assessed by estimating the ratio between the number of gametocytes and the number of infected red blood cells.

Oocyst test: Effect on the infectivity of gametocytes. One hour after having received drugs, mice with infected red blood cells are narcotized and placed, individually, over cages containing Anopheles stephensi females (for 45 min.). Ten days after the blood meal, mosquitoes are dissected and their mid-guts examined under the light microscope to assess oocyst presence and count their number (that means that gametocytes in infected blood were able to develop a sexual cycle in the mosquito gut).

Sporozoïte test: Sporonticidal activity (Prophylactic activity – inhibition of the infection by sporozoites). Mice are treated before (x days before) infection by sporozoites. Mice are then challenged by exposing them to infectious mosquito bites on day 0. Parasite density is determined on Giemsa-stained thin blood smears, sampled from the tail of the mice at day 5,6,7… In some cases mice are directly infected by injection of suspension of sporozoites extracted from mosquito’s salivary glands. Drugs are administrated 2 hours after infection by sporozoites. Smears are made 48 hours after infection and every day until parasitaemia reaches 2%.

Liver stages: After administrating sporozoïtes, it is especially important to distinguish between genuine activity against the liver schizonts and action on first generation merozoites emerging from them; this is especially likely with a compound that possesses a significant repository action. However, pre-erythrocytic schizonts can also be sought directly in Giemsa-stained microscope sections of the livers of treated and control animals, but only if they receive exceptionally heavy sporozoite inocula. To differentiate between schizonticid activity (red blood stages) in the case of a long lasting compound and prophylactic activity (sporozoites and liver stages), groups infected with erythrocytic stages are added in the experiment and time to reach 2% is compared with sporozoites infected groups.

3. Toxicity

The median lethal dose, LD50 (abbreviation for "lethal dose, 50%"), LC50 (lethal concentration, 50%) or LCt50 is a measure of the dose required to kill half the members of a tested population after a specified test duration.


B. IN VITRO
1. Schizontocidal activity (activity on erythrocytic stages)

Erythrocyts: Classically drugs are added to P. falciparum cultured in red blood cells and the activity of the drugs is determined by isotopic ([3H]-hypoxanthine), enzymatic (LDH), fluorometric (syber green) or visual (microscope) methods.

Liver stage: Drugs are tested on human or rodent species of Plasmodium. Primary human (HH) or murine hepatocytes (MH), are infected with murine malaria or P. falciparum sporozoites. Compounds are added simultaneously with sporozoites at different concentrations. The culture medium is then replaced 3h after sporozoites inoculation and then on a daily basis. The inhibitory effect of the compound is quantified, after fluorescent immunostaining of the liver-schizonts with a mouse serum raised against PfHSP70 through fluorescence microscopic analysis (some variations are also described in the literature).

2. Transmission inhibition test

Gametocyte: P. falciparum cultures are submitted to different types of stimulation to generate gametocytes and gametocytes are counted with an optical microscope (some variations are also described in the literature) after the drug challenge. Sporozoites: They are collected from salivary glands and challenged with the drugs prior infecting hepatocytes (see liver stage).

3. Toxicity

Many Cell types can be used to screen for toxicity both by estimation of the basal functions of the cell or testing specialized cell functions. SI refers to the ratio of the toxic dose of the drug (or TC50) divided by the dose that leads to the desired pharmacological effect (IC50 in Plasmodium). The higher the SI, the higher is the selective antiplasmodial activity of a given extract or compound.
